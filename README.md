# Assembler for the 16-bit Hack Assembly language in Python

**Run the assembler:**
Pick a .asm file from ../hack-examples dir
```shell 
python3 run.py simple_add.asm 
0000000000000010
1110110000010000
0000000000000011
1110000010010000
0000000000000000
1110001100001000
``` 
You can find the binary executable at "../testing/simple_add_actual.hack". Now you can run it on hack hardware.
